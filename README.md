## プロジェクト

現在進行中のプロジェクトや過去の作品を紹介します。

### ネズミスケッチ

[ネズミスケッチ](https://mouse-sketch.web.app/)

![ネズミスケッチのテキスト](https://mouse-sketch.web.app/img/nezumisuketitext.png)

![ネズミスケッチのスクリーンショット](https://mouse-sketch.web.app/assets/%E3%82%B9%E3%82%AF%E3%83%AA%E3%83%BC%E3%83%B3%E3%82%B7%E3%83%A7%E3%83%83%E3%83%88-BBBS5w4E.png)

ネズミスケッチは、ネズミをテーマにしたドット絵アプリです。ユーザーは自由にネズミのドット絵を描くことができます。アニメーションやレイヤー機能もあり、保存・共有が可能です。

### Inpaint-nezumi

[Inpaint-nezumi](https://mouse-sketch.web.app/Inpaint-nezumi/)

![Inpaint-nezumiのスクリーンショット](/nezumi-inpaint.png)

Inpaint-nezumiは、画像の不要な部分を消すことができるツールです。ユーザーは消したい部分を選択し、AIが自然な背景で埋めてくれます。写真の修正やオブジェクトの削除に便利です。  
現在はベータ版で、今後さらに機能を追加していく予定です。  
これはossプロジェクトです。アパッチライセンスの下で公開されています。
